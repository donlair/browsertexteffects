From beams and fireworks to matrix rain and VHS distortion — a rich library of animated text effects ready to use out of the box.

◎

Zero Dependencies

Pure TypeScript compiled to a single bundle. No runtime dependencies. Works with any framework or plain HTML — just import and animate.

⟨/⟩

Simple API

Two functions cover 90% of use cases. Full config control for every effect, plus scroll-triggered animations built in with createEffectOnScroll.

Quick Usage

import { createEffect } from "browsertexteffects";

const el = document.getElementById("my-text");
const handle = createEffect(el, "Hello, World!", "beams");
handle.start();

// Stop at any time:
handle.stop();
